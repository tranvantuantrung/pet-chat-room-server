# Realtime Chat Room server

Realtine Chat Room Server is the back end of the pet project app.

## Installation

User NodeJS + Socket.io library on the back end.

Setup:

```bash
npm install
```

##Usage

Run app:

```bash
cd pet-chat-room-server
node src/index.js
```
